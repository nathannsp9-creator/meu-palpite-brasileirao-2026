# meu-palpite-brasileirao-2026

Projeto de exemplo para o Bolão do Brasileirão.

## Rodando localmente

Pré-requisitos: `Node.js` e `npm`.

```powershell
git clone <YOUR_GIT_URL>
cd <YOUR_PROJECT_NAME>
npm install
npm run dev
```

## Tecnologias

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Deploy

Build de produção:

```powershell
npm run build
npm run preview
```

Substitua ou configure o deploy conforme seu provedor de hospedagem (Vercel, Netlify, etc.).
